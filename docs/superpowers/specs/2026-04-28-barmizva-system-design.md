# מערכת בר המצווה — דון בר אל
**תאריך:** 2026-04-28  
**סטטוס:** מאושר

---

## סקירה כללית

מערכת ווב לניהול הזמנות לבר המצווה של דון בר אל. המערכת מאפשרת לאב לנהל רשימת מוזמנים, לעצב הזמנה, לשלוח הודעות WhatsApp אוטומטיות עם קישור RSVP אישי לכל מוזמן, ולעקוב אחר אישורי ההגעה בזמן אמת.

---

## ארכיטקטורה

```
פאנל ניהול (Vercel /admin)
  → whatsapp-web.js (סקריפט מקומי על מחשב המשתמש)
    → WhatsApp המוזמן (הודעה + קישור RSVP ייחודי)
      → דף RSVP (Vercel /rsvp/[token])
        → Supabase (שמירת תשובה)
          → פאנל ניהול (עדכון סטטוס)
```

**מחסנית טכנולוגית — הכל חינם:**
- **Next.js 14** (App Router) — פרונטאנד + API routes
- **Vercel** (free tier) — הוסטינג ודפלוי
- **Supabase** (free tier) — PostgreSQL לנתונים
- **whatsapp-web.js** (open source) — שליחה אוטומטית מהמחשב
- **Tailwind CSS** — עיצוב

## דפים ונתיבים

### `/admin` — פאנל ניהול
מוגן בסיסמה. מכיל 4 לשוניות:

**דשבורד:**
- 4 כרטיסי סטטיסטיקה: מגיעים / לא מגיעים / לא בטוח / ממתינים
- רשימת מוזמנים עם סטטוס צבעוני
- סינון לפי סטטוס

**ניהול מוזמנים:**
- הוספת מוזמן (שם + טלפון)
- עריכה ומחיקה
- ייבוא CSV (אופציונלי)

**עורך הזמנה:**
- שדות עריכה לכל פרטי האירוע
- תצוגה מקדימה של דף RSVP בזמן אמת
- עריכת תבנית הודעת WhatsApp

**שליחה:**
- בחירת מוזמנים לשליחה (כולם / ממתינים / לא בטוח)
- כפתור "שלח הזמנות" — מייצר קובץ JSON ומציג הוראות להרצת הסקריפט
- כפתור "שלח תזכורת ל-לא בטוח" — אותו תהליך עם הודעה שנייה

### `/rsvp/[token]` — דף RSVP
- עיצוב בוטני (לבן + עלים ירוקים SVG בפינות)
- שם המוזמן מוצג אישית ("שלום, [שם] ❤")
- פרטי האירוע המלאים
- 3 כפתורים: **מגיע בשמחה** / **לא בטוח** / **לא אוכל להגיע**
- לחיצה שומרת תשובה ב-Supabase + ממנה לדף תודה מתאים

### `/confirmed/[token]` — דף אישור "מגיע"
- הודעת תודה חמה + פרטי האירוע מסוכמים
- עיצוב בוטני תואם

### `/maybe/[token]` — דף "לא בטוח"
- הודעה: "קיבלנו! ניצור קשר לאישור סופי 💛"
- הסבר שתגיע תזכורת בוואטסאפ
- עיצוב בוטני תואם

### `/declined/[token]` — דף "לא מגיע"
- הודעת תודה מנומסת
- עיצוב בוטני תואם

## סקריפט WhatsApp מקומי (`scripts/send-whatsapp.js`)

סקריפט Node.js שרץ על מחשב המשתמש:

1. מתחבר ל-Supabase וקורא את רשימת המוזמנים הרלוונטיים
2. מאתחל whatsapp-web.js → מציג QR בטרמינל לסריקה חד-פעמית
3. שולח לכל מוזמן הודעה בנוסח:
   ```
   שלום [שם] 🎉
   [נוסח חופשי מהפאנל]
   לאישור הגעה לחץ כאן:
   https://barmitzva-don.vercel.app/rsvp/[token]
   ```
4. מעדכן `reminder_sent = true` ב-Supabase
5. לוג מובנה: כמה נשלחו, כמה נכשלו

**מצב `--mode reminder`:** שולח הודעה שנייה רק למוזמנים עם `status = maybe`:
```
שלום [שם] 🙏
טרם קיבלנו אישור הגעה לבר המצווה של דון.
נשמח לדעת בהקדם — לחץ לאישור:
https://barmitzva-don.vercel.app/rsvp/[token]
```

## מסלול המשתמש (סיכום)

1. כניסה ל-`/admin` → מוסיף מוזמנים (שם + טלפון)
2. עריכת פרטי ההזמנה בעורך
3. מריץ `node scripts/send-whatsapp.js` → סורק QR → כולם מקבלים WhatsApp
4. מוזמנים לוחצים קישור → בוחרים סטטוס → נשמר ב-Supabase
5. "לא בטוח" → דף מיוחד + פאנל מסמן אותם
6. מריץ `node scripts/send-whatsapp.js --mode reminder` → תזכורת נשלחת

## אבטחה

- פאנל ניהול מוגן בסיסמה (env var `ADMIN_PASSWORD`)
- כל קישור RSVP כולל טוקן UUID אקראי — לא ניחוש
- API routes ב-Vercel לא חושפות נתוני מוזמנים ללא טוקן תקין
- Supabase Row Level Security מופעל
